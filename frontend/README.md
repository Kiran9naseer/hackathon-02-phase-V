# Todoist Master - Neural Interface

A premium, state-of-the-art task management dashboard with a dark "Mist Green" aesthetic.

## 🚀 Advanced Features

- **Unified Command Palette (`Ctrl+K`)**: Instant access to global search, task creation, and system actions.
- **Neural Notifications**: Real-time event streaming via SSE with premium toast alerts for task updates and reminders.
- **Protocol Automation**: Full support for recurring task series with dedicated management interface.
- **Strategic Reminders**: Intelligent deadline alerts with multi-offset configurations.
- **Visual Intelligence**: Glassmorphic cards, priority-coded badges, and dynamic progress matrices.

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with custom Mist-Green design system
- **State Management**: React Hooks & Context API
- **Icons**: Lucide React
- **Authentication**: JWT-based session management

## 🎨 Aesthetics

- **Dark Mode First**: Optimized for low-light high-productivity environments.
- **Glassmorphism**: High-fidelity background blurs and translucent surfaces.
- **Micro-animations**: Smooth transitions and interactive feedback for a premium feel.

## 🏁 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
